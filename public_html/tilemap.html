Constructor

new TileMap (xPos, yPos, tileWidth, width, height)

Constructor that creates a TileMap

xPos: The world coordinate x position of the TileMap
yPos: The world coordinate y position of the TileMap
tileWidth: The world coordinate width of each tile
width: The width of the TileMap in tiles
height: The height of the TileMap in tiles

Methods

getWCHeight ()

Returns the world coordinate height of the TileMap


getWCWidth ()

Returns the world coordinate width of the TileMap


getWidth ()

Returns the width of the TileMap in tiles


getHeight ()

Returns the height of the TileMap in tiles


getXPos ()

Returns the world coordinate x position of the TileMap


getYPos ()

Returns the world coordinate y position of the TileMap


clearMap ()

Removes all tiles from the TileMap


addTile (tileX, tileY, renderable)

Adds a renderable to the TileMap

tileX: The target x for the new tile
tileY: The target y for the new tile
renderable: The renderable to be put in the TileMap


removeTile (tileX, tileY)

Removes a tile from the TileMap

tileX: The target x to remove from
tileY: The target y to remove from


draw (camera)

Draws the TileMap

camera: The camera to draw the TileMap on


isTileAt (xPos, yPos)

Returns true if tile is at given x and y

tileX: The target x to check for a tile
tileY: The target y to check for a tile


getBounds (xPos, yPos)

Returns a bounding box for the tile at given x and y

tileX: The target x to get bounding box
tileY: The target y to get bounding box
