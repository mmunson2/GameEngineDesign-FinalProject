Constructor

Constructor that creates a ShapeGen object

Methods

rectangle (x, y, height, width, color)

Creates a colored rectangle at given tile x and y

x: The tile x to put the rectangle at
y: The tile y to put the rectangle at
height: The height in tiles of the rectangle
width: The width in tiles of the rectangle
color: The color array for the tiles to be colored


texturedRectangle (x, y, height, width, texture, color, UVArray)

Creates a textured rectangle at given tile x and y

x: The tile x to put the rectangle at
y: The tile y to put the rectangle at
height: The height in tiles of the rectangle
width: The width in tiles of the rectangle
texture: The texture file to use
color: The color array for the tiles to be colored
UVArray: The UV coordinates the specify the texture

circle (xc, yc, r, color, fill)

Creates a colored circle centered at given tile x and y

xc: The tile x to put the circle at
yc: The tile y to put the circle at
r: The radius of the circle
color: The color array for the tiles to be colored
fill: True for filled in circle, false for empty circle

_drawCircle (xc, yc, x, y, color)

Private helper function that draws the colored circle

y: The calculated y offset
color: The color array for the tiles to be colored
